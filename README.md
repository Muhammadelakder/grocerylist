# تطبيق قائمة المشتريات
> . باستعمال بايثون وجافاسكربت


### تعليمات التنزيل

1. **تأكد من اتصالك بالإنترنت**

2. **`Python3` نزل**

3. **`pip` نزل**

4. **نزل متطلبات عمل التطبيق**
    `pip install -r requirements.text`

## تعليمات البدء

2. **ابدء التطبيق باستخدام الامر التالي**
    `python manage.py runserver`
